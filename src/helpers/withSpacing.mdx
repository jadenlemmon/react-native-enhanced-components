# With Spacing HOC

## Minimel Example

The With Styles HOC provides a spacing wrapper that can wrap any component and provide spacing utilility props.

<Playground>
  {() => {
    const MyCustomButton = withSpacing(TouchableOpacity);
    return (
        <MyCustomButton p={50} style={{ backgroundColor: 'blue' }}>
          <Text style={{ color: 'white', textAlign: 'center' }}>My Button</Text>
        </MyCustomButton>
    );
}}

</Playground>
